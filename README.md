# 🔐 Flask Authentication System

A simple user authentication system built with Flask.  
It supports user registration, login, logout, and protected routes with session management.

---

## 🚀 Live Demo on deployed using Render

👉 https://flask-authentication-9ldb.onrender.com

---

## 📌 Features

- User registration with form validation
- Secure password hashing (Werkzeug)
- User login/logout system
- Protected routes using Flask-Login
- SQLite database integration
- Flash messages for user feedback
- File download (protected route)

---

## 🧰 Tech Stack

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login
- Flask-WTF
- WTForms
- SQLite
- HTML/CSS (Jinja templates)
